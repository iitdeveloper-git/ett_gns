# Admin onboarding

Use the admin console topbar selectors; do not paste hidden tenant or application IDs into forms.

1. Create or select a tenant.
2. Create or select an application.
3. Register an event such as `user.welcome`.
4. Create a template draft, validate it, preview it, then publish it.
5. Register a provider only after `Test connection` succeeds.
6. Create an application credential and store the secret in a secret manager.
7. Send a test notification and inspect the operations timeline.

The dashboard checklist shows completion status for these steps and links to the next screen.
