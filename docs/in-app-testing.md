# In-App Testing

Covered locally:

- Template validation
- In-app runtime acceptance
- Worker storage
- Notification center listing
- Unread count
- Ack/displayed
- Mark read
- Dismiss
- Preferences
- Deduplication conflict
- Cross-tenant isolation
- Existing channel regression suite

Load test artifacts are runnable in `load-tests/in-app-sse.js`; actual results require `k6` and a representative deployment.
